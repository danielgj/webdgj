Sobre esta web

Mi actual proyecto consiste en aprender algún framework moderno de desarrollo web. Creo que apostaré por React.

En ese afan, he decido crear esta página web para ir poco a poco volcando en ella mis nuevos conocimientos. Hasta que llegue ese momento si he plasmado aquí algunas "innovaciones" en mi forma de trabajar.

Como resumen listo aquí algunas de las cosas que estoy haciendo (de momento) en esta web:

  • Github: He subido el código a un repositorio público y lo licenciaré con alguna licencia Open Source para quue quien quiera pueda coger el código y usarlo, extenderlo, etc.
  • Template Boostrap a pelo: Lo que explica porque es tan fea ;) La idea es crear un template de Boostrap 4 desde cero que sea 100% responsive y se comporte adecuadamente en cualquier tipo de pantalla.
  • Uso de Saas para la edición de las hojas de estilo
  • Uso de Gulp para automatización de tareas del desarrollo: De momento tengo "live edit", compilado de CSS, generación de scripts, creación de versión "dist", "minificado" y "uglificado" y hasta ejecución de tests (de momento inexistentes).

Los siguientes pasos serán:

  • Babel para poder probar y aprender ECMA6
  • Testing
  • React o uso de otro FWK
  • Multi-idioma.
